Koba Museum - Vertical and Horizontal Interactive Touch Panels


Description:
The creative work at Koba Ashram (Museum) was designed through interactive display screens and audio to showcase the life journey of the Saints (Priests) from Gujarat (India) who went on various pilgrimage continuously.


The interactive touch screens showcase their journey and the activities involved. Image carousel and scroll content were created for the ease of user to interact with the content.


The application for touch displays had multi-lingual content (in English and Gujarati languages).

Igloo Cave - 360 degree immersive space

Overview

This project was executed at Igloo 360 Immersive Space at University of Southampton. Different types of fountains in the city of Southampton were 3D scanned initially, then were cleaned and optimized using 3D softwares like Reality Capture and Blender.


Later these were imported in Unity 3D game engine and created windows desktop based executable using Igloo Plugin.


The complete project was 360 degree immersive, and the user standing at the center of this setup controls the movement of camera using the tablet provided. The user is able to digitally go near to each 3D models and study the architecture details, and learn each details of the fountains.
