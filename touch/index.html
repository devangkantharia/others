Marker Detection Touch Table

Overview

The task was to create a multi user collaborative application using latest technology.


The solution designed was a marker based touch application and software, where different markers signify different sectors and sub-processes.


If the markers are kept on the table top surface, the cameras below it, detects the respectve QR code stickers attached to each physical marker.


Users can also touch the cards created, and connect them by drawing lines between them to show links.

Siemens Elecrama - Expo

Overview

Siemens Elecrama - Expo


Siemens Elecrama is an event that is conducted every 2 years, where Siemens leverages latest technology to showcase their unique products and services.


In this project, a LED immersive cave dome was created with 8 touch panels, that refers to 8 sector of their new product segments.


The complete visual of the LED as well as the touch panel was synchronized in real-time in the backend. User can interact with the touch screens and explore the product features and services offered by Seimens.
