Cycle Simulator

Overview

Multi player phygital game where three or less people can compete eachother by racing with a bicycle.

For more immersive experience, user can wear VR headset.

Multiple iterations were done over 5 months to come to final stage of delivery.

Responsibility

  • Tech coordinator
  • UX designer
  • Assistant Project Manager
  • Prototype designer and developer
  • Shader Programmer

Soccer Simulator

Overview

In this project, the user selects a scenario which is fetched from live FIFA scenario fetching data through their API, and then user is asked to take 3 shots to achieve same like the fifa player choosen.

Kit used checks ball position in realtime and give analyses at the end of the gameplay.
