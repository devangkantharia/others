Object / Surface Tracking

Overview

In this project, two options were given to the end user to choose from:


Model/Object Tracking: In this mode, the user will be able to augment the complete digital version of the city model on top of real architectural physical model, which than shows interaction points and story line.


Surface Tracking: In this mode, the user will choose a flat surface/floor and then after backend calculations, the complete city model shows up on the floor which can later be interacted with to experience the content.

Responsibility

  • Tech coordinator
  • UX designer
  • Assistant Project Manager
  • Prototype designer and developer
  • Unity3D developer
  • Shader Programmer
